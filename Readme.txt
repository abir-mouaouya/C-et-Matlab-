*********************************** MINI PROJET 2 : INTERFACE GRAPHIQUE MATLAB : FFT Matlaab  VS   FFT C++ ******************************************

Le deuxième mini projet de C++,  a comme objectif la comparaison entre  la FFT générer par un script MATLAB, 
et la FFT calculée par un programme en C++ avec la formule générique de la TFD, afin de comparer les valeurs stockées dans deux fichiers différents par 
le logiciel WinMerge.
